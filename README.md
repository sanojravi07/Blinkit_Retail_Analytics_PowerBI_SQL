# Blinkit Retail Analytics Dashboard (SQL + Power BI)

## Project Overview
This project analyzes Blinkit retail data using SQL and Power BI to uncover insights related to sales performance, customer behavior, product demand, and operational pain points.

## Tools Used
- SQL
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Dashboard Pages
1. Executive Overview  
2. Product Insights  
3. Customer Insights  
4. Pain Points & Opportunities

## Key Insights
- Identified top and bottom performing products
- Measured category-wise revenue contribution
- Analyzed customer segment performance
- Highlighted delivery delay problem areas
- Created KPI to track focus zones below average revenue
